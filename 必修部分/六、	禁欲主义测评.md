## 2、吃饭
定义9：吃饭指数（the index of dining，符号为x吃）计算方法：  
```python
#t的单位为min
def x(t):
  if t>=0 and t<=15:
    return 5-t/5
  elif t>15:
    return 1
```
其中，当不同饭菜差异可以忽略时（抓主要矛盾），t吃是吃饭速率的积分上限，该积分则等于饭菜总质量。
